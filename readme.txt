Powerful plugin to add icon support to a known custom fields plugin. From developer to developer.

== Description ==

This plugin came from a need, it was built for personal use but, I decide to publish it and hope it can be useful for somebody else.

=== Support/Issues ===
Please use the either the [support](https://wordpress.org/support/plugin/kin-icon-picker/) forum or the [official repository](https://github.com/plovinicius/kin-icon-picker) for any questions or to log issues.

== Screenshots ==

1. Settings, upload the .zip file with the font icons downloaded from Icomoon site.
2. If uploaded with success, you will see a list below the upload btn, with the font files.
3. Create a new field.
4. Select an icon from the list.
